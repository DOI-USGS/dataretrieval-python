Welcome

Welcome to the documentation for the Python dataretrieval package. dataretrieval is a Python alternative to the USGS R dataRetrieval package and is used to obtain USGS and EPA water quality data, streamflow data, and metadata directly from webservices (see the data portals documentation for additional details about specific data sources).
